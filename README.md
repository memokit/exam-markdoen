# รายการ (List)

## แบบไม่เรียงลำดับ (Unordered)

```md
+ การสร้างรายการแบบไม่เรียงลำดับให้ขึ้นต้นบรรทัดด้วยเครื่องหมาย `+`, `-`, หรือ `*`
*  การสร้างรายการย่อยให้เคาะเว้นวรรค 2 ครั้ง 
  - เครื่องหมายด้านหน้าจะถูกเปลี่ยนก็ต่อเมื่อมีการขึ้นรายการใหม่
    * รายการที่ 1
    + รายการที่ 2
    - รายการที่ 3
+ รายการใหม่
```
